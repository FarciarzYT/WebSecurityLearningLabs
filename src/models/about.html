What is this?

Web Security Learning Labs is a hands-on web security education project created by FarciarzYT for the Hack Club “Flavortown” competition.

The goal is to help people understand how real-world web vulnerabilities happen and how to think about security the right way.

Hack Club Flavortown

Why?

Because beginners often learn web development without learning security. This project exists to make web security approachable, practical, and honest.
