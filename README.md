# Web Security Learning Labs

Interactive web platform to learn and practice web security in a safe environment. Designed for beginners and intermediate users, it provides hands-on labs, challenges, and tutorials.

## Demo

Check out the live demo: [https://web-security-learning-labs.vercel.app/](https://web-security-learning-labs.vercel.app/)

## Features

- Fully responsive website design
- Interactive hacking labs and exercises
- Tutorials to understand common web vulnerabilities
- Safe environment to practice web security techniques
- Modern UI for an engaging learning experience

## Getting Started

### Prerequisites

- React (v18+ recommended)
- npm
